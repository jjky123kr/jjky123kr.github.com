## HTML 페이지 구조

```````
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
</head>
<body>

</body>
</html>
`````````

* charset ="UTF-8" 으로 설정 해야 한다. 
-> 한글이 깨지지 않기 위해서 
* 단축키    
  html 주석 단축키 :Ctrl+shift+/  
  html 주석 해제:Ctrl+shift+\  
  Ctrl+Alt+ 아래 방향키: 복사
