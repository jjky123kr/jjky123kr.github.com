### http 통신 : 약속

* 클라이언트 가 서버에 요청 할때 사용한다. 

#### 통신 요청 방법 4가지

##### get : 데이터를 요청 (select)

#### post 방식 : 데이터를 서버에 추가  (insert)

#### put  방식 : 데이터 수정 (update)

#### delete  방식 : 데이터 삭제 (delete)
